# LiveShop - Application de Shopping en Direct

Une application Flutter moderne pour le shopping en direct avec chat en temps réel, gestion de panier et filtrage par catégories.

## Fonctionnalités

- Événements en direct : Regardez des streams de shopping en temps réel
- Chat interactif : Communiquez avec les vendeurs et autres spectateurs
- Panier intelligent : Ajoutez des produits avec variations (taille, couleur)
- Filtrage par catégories : Mode, Beauté, Électronique
- Profil utilisateur : Consultez vos commandes et notifications
- Design moderne : Interface élégante avec animations fluides

## Captures d'écran

L'application comprend :
- Écran d'accueil avec événements live, à venir et replays
- Vue détaillée des événements avec vidéo et chat
- Panier avec gestion des quantités
- Profil avec historique des commandes

## Installation

Consultez le fichier INSTALL.md pour les instructions d'installation détaillées.

## Technologies utilisées

- Flutter : Framework UI multiplateforme
- Provider : Gestion d'état
- GoRouter : Navigation
- Google Fonts : Typographie (Sora)
- json_serializable : Sérialisation JSON

## Structure du projet

```
lib/
├── config/          # Configuration (thème, routes)
├── models/          # Modèles de données
├── providers/       # Providers pour la gestion d'état
├── screens/         # Écrans de l'application
├── services/        # Services (API, WebSocket)
├── widgets/         # Widgets réutilisables
└── main.dart        # Point d'entrée
```

## Design

L'application utilise un design moderne avec :
- Couleur primaire : Rouge vif (#FF2600)
- Police : Sora (Google Fonts)
- Thème clair et sombre
- Animations fluides et micro-interactions
